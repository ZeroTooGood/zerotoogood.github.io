About Gourmet Getaway

Gourmet Getaway is a student-built demonstration website inspired by modern food-tech apps like Zomato. This project was created as a hands-on learning exercise to practice responsive web design, analytics (Google Analytics) setup, SEO basics (meta tags and alt text), social media integration, and simple e-commerce flows such as menus, cart management, coupon application and checkout options. The site is intended for educational use — it demonstrates UI/UX patterns, user flows, and tracking practices so students can understand how front-end interfaces connect with analytics and digital marketing strategies. Use the search, filters, and menus to explore how a food ordering front-end behaves.
